## 개요
**3D 얼굴 골격(구조) 분석**과 **멀티모달 생성**을 결합해 **개인 맞춤형 화장/뷰티 추천**을 제공하는 에이전트입니다. 개인의 얼굴 기하 구조를 바탕으로 추천을 개인화합니다.
